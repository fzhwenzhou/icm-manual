# Preface

This document describes the ICM basic architecture. This is the first version of this manual.

The modules marked *Ratified* have been ratified at this time. The modules marked *Frozen*
are not expected to change significantly before being put up for ratification. The modules marked
*Draft* are expected to change before ratification. The modules marked *Planning* are in schedule but not presented yet, and can be deleted in further releases.
