# Standard Dense Modules

- 1 M - Integer Multiplication Module

- 2 F - Floating Point Module
        
        2.1 F32 - IEEE 754 32-bit Floating Point

        2.2 F64 - IEEE 754 64-bit Floating Point

        2.3 F16 - IEEE 754 16-bit Floating Point

- 3 V - Vector Module

        3.1 Vs - Small Vectors

        3.2 Vl - Large Vectors

- 4 Ma - Matrix Module

        4.1 Mas - Small Matrices

        4.2 Mal - Large Matrices
